# RAXY Pooling

RAXY Pooling is a complete and lightweight pooling system to boost your Unity project's performance.

Add the **Pool Manager** to any GameObject in your scene, then add **PoolableObject** or **PoolableParticleSystem** to any prefab you want to pool.

**Note:** Odin Inspector is used to visualize the pool list in the editor, but the system works without it.
